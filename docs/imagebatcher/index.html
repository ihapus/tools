ImageBatcher

Batch resize and compress images on Windows for e-commerce workflows, social media prep, and routine JPG or PNG cleanup.

ImageBatcher is built for users who regularly need to resize images to specific dimensions, lower file size, and export processed files in one pass without opening an editor image by image.

What ImageBatcher helps with

Batch resize images

Resize multiple files to preset or custom dimensions without editing each file individually.

Prepare listings faster

Useful for product images, marketplace uploads, and content libraries that need consistent output.

Main features

  • Batch import images and folders
  • Preset and custom width and height settings
  • Quality adjustment for processed output
  • Progress feedback during processing
  • Output directory selection
  • Result log after processing completes
