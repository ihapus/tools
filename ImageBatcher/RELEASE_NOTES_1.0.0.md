# ImageBatcher 1.0.0

## What it does
ImageBatcher is a Windows batch image resizer and compressor for JPG and PNG workflows.

## Good for
- batch resize images on Windows
- compress JPG and PNG files before upload
- prepare product listing images
- standardize repeated export workflows

## Included in this release
- `ImageBatcher-1.0.0-setup.exe`
- `ImageBatcher-1.0.0-setup.sha256.txt`
- `LICENSE`
- `THIRD_PARTY_NOTICES.txt`

## System requirements
- Windows 10 or later
- Enough disk space for input and output images
