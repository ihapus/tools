# Release template

Use this format for GitHub Releases in the `tools` repository.

## Title format

```text
ImageBatcher 1.0.0 - Windows batch image resizer and compressor
```

## Body template

```markdown
## What it does
ImageBatcher is a Windows batch image resizer and compressor for JPG and PNG workflows.

## Good for
- batch resize images on Windows
- compress JPG and PNG files before upload
- prepare product listing images
- standardize repeated export workflows

## Download
Download the installer from the release assets attached to this page.

## System requirements
- Windows 10 or later
- Enough disk space for input and output images

## Included files
- installer executable
- LICENSE
- THIRD_PARTY_NOTICES.txt

## Notes
- This release is distributed as proprietary software by ihapus.
- Third-party components remain under their own license terms.
```
